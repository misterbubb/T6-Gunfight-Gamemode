## Installation

Run `install_gunfight_sd.ps1` or manually copy files:
- `gunfight_mp` folder → `%localappdata%\Plutonium\storage\t6\mods\`
- `sd.gsc` → `%localappdata%\Plutonium\storage\t6\raw\maps\mp\gametypes\`
